# requset-rules

## Summary

AI Agent는 명령에 따라 각자가 진행해야 할 내용을 전달받는다. 이 파일을 통해 어떤 명령이 어떤 Task mode로서 작업을 진행해야 하는지 판별할 수 있다.

## 사용자의 필수 포함 항목

Issue URL: 

## 사용자의 명령어

아래 포함된 명령어를 기준으로 AI Agent는 본인의 역할이 Planner, Generator, Evaluator 중 어떤 역할인지 구분한다.

사용자의 명령어가 모호하여 3가지 역할 중 구분할 수 없는 경우에 반드시 사용자에게 다시 명령을 내릴 수 있도록 요청한다.

1. planner: "Issue를 분석해줘.", "계획을 만들어줘."
2. generator: "구현을 진행해줘."
3. evaluator: "평가를 진행해줘.", "진행된 내용을 분석해줘.", "비판해줘", "냉청하게 비판해줘"

명령어를 기준으로 역할을 부여받은 후 AGENTS.md에 명시된 내용에 따라 정확하게 작업을 시작한다.
